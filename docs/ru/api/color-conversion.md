# Преобразование цветов

Утилиты для парсинга и преобразования цветов: разбор hex (3/4/6/8 цифр), CSS-строки `rgb()`/`rgba()`, двустороннее преобразование HSL и универсальная функция приведения `toRgb()`.

Они экспортируются как для прямого использования, так и потому, что высокоуровневые API ([`simulate()`](./simulate), [`simulatePalette()`](./simulate-palette), [вспомогательные функции доступности](./accessibility)) принимают цвет в любом формате, пропуская его через `toRgb()`.

---

## toRgb()

Приводит цвет в любом поддерживаемом формате к RGB-объекту. Универсальная точка входа, используемая остальным API.

```ts
function toRgb(color: string | RGB | HSL): RGB
```

Принимает hex-строки (`"#e63946"`, `"F00"`, `"#RRGGBBAA"`), CSS-строки `rgb()`/`rgba()`, HSL-объекты или RGB-объекты (возвращается как свежая копия).

### Примеры

```ts
import { toRgb } from 'chromanopia'

toRgb('#ff0000')                  // { r: 255, g: 0, b: 0 }
toRgb('rgb(230 57 70)')           // { r: 230, g: 57, b: 70 }
toRgb({ h: 120, s: 1, l: 0.5 })   // { r: 0, g: 255, b: 0 }
toRgb({ r: 10, g: 20, b: 30 })    // { r: 10, g: 20, b: 30 } (copy, not alias)
```

::: tip
RGB-объекты возвращаются как копии, никогда не та же ссылка, поэтому изменение результата безопасно.
:::

## parseCssColor()

Разбирает CSS-строку цвета в RGB-объект. Более низкоуровневая, чем `toRgb()` — принимает только строки.

```ts
function parseCssColor(css: string): RGB
```

Поддерживает hex (3/4/6/8 цифр, с `#` или без) и `rgb()` / `rgba()` как в устаревшем синтаксисе с запятыми (`rgba(230, 57, 70, 0.5)`), так и в современном синтаксисе с пробелами (`rgb(230 57 70 / 0.5)`). Каналы в процентах (`rgb(100%, 0%, 0%)`) поддерживаются. Компонент альфа разбирается, но не возвращается (у `RGB` нет альфа-канала).

### Примеры

```ts
import { parseCssColor } from 'chromanopia'

parseCssColor('#e63946ff')              // { r: 230, g: 57, b: 70 } (8-digit, alpha dropped)
parseCssColor('rgb(230, 57, 70)')       // { r: 230, g: 57, b: 70 }
parseCssColor('rgba(230 57 70 / 0.5)')  // { r: 230, g: 57, b: 70 }
parseCssColor('rgb(100%, 0%, 0%)')      // { r: 255, g: 0, b: 0 }
```

Бросает исключение при неподдерживаемых форматах (именованные цвета вроде `"red"`, строки `hsl()` и т.д. не поддерживаются — для HSL сначала преобразуйте через `hslToRgb()`).

## hexToRgb()

Разбирает hex-строку цвета в RGB-объект.

```ts
function hexToRgb(hex: string): RGB
```

Принимает `#RGB`, `#RGBA`, `#RRGGBB`, `#RRGGBBAA` (с ведущим `#` или без него). Компонент альфа разбирается, но отбрасывается.

### Примеры

```ts
import { hexToRgb } from 'chromanopia'

hexToRgb('#e63946')   // { r: 230, g: 57, b: 70 }
hexToRgb('e63946')    // { r: 230, g: 57, b: 70 } (# optional)
hexToRgb('#F00')      // { r: 255, g: 0, b: 0 }   (3-digit)
hexToRgb('#e63946ff') // { r: 230, g: 57, b: 70 } (8-digit, alpha dropped)
```

Бросает `Error` при некорректном вводе — проверяется вся строка целиком, поэтому частично корректные строки вроде `#12gggg` отклоняются (без скрытого парсинга префикса через `parseInt`).

## rgbToHex()

Преобразует RGB-объект в 6-значную hex-строку с префиксом `#`.

```ts
function rgbToHex(rgb: RGB): string
```

Значения каналов обрезаются до диапазона [0, 255] и округляются.

### Примеры

```ts
import { rgbToHex } from 'chromanopia'

rgbToHex({ r: 230, g: 57, b: 70 }) // '#e63946'
rgbToHex({ r: 0, g: 0, b: 0 })     // '#000000'
rgbToHex({ r: 300, g: -5, b: 0 })  // '#ff0000' (clamped)
rgbToHex({ r: 127.6, g: 0, b: 0 }) // '#800000' (rounded)
```

## rgbToHsl()

Преобразует RGB-цвет в HSL.

```ts
function rgbToHsl(rgb: RGB): HSL
```

Возвращает `{ h, s, l }`, где `h` — тон в градусах [0, 360), а `s`/`l` — насыщенность и светлота как доли в диапазоне [0, 1].

### Примеры

```ts
import { rgbToHsl } from 'chromanopia'

rgbToHsl({ r: 255, g: 0, b: 0 })   // { h: 0,   s: 1, l: 0.5 } (red)
rgbToHsl({ r: 0, g: 255, b: 0 })   // { h: 120, s: 1, l: 0.5 } (green)
rgbToHsl({ r: 0, g: 0, b: 255 })   // { h: 240, s: 1, l: 0.5 } (blue)
rgbToHsl({ r: 128, g: 128, b: 128 }) // { h: 0, s: 0, l: 0.5 } (gray → s: 0)
```

---

## hslToRgb()

Преобразует HSL-цвет в RGB.

```ts
function hslToRgb(hsl: HSL): RGB
```

Выходящее за пределы диапазона значение `h` нормализуется по модулю 360; `s` и `l` обрезаются до [0, 1].

### Примеры

```ts
import { hslToRgb } from 'chromanopia'

hslToRgb({ h: 0,   s: 1, l: 0.5 }) // { r: 255, g: 0,   b: 0   } (red)
hslToRgb({ h: 120, s: 1, l: 0.5 }) // { r: 0,   g: 255, b: 0   } (green)
hslToRgb({ h: 480, s: 1, l: 0.5 }) // { r: 0,   g: 255, b: 0   } (480° ≡ 120°)
hslToRgb({ h: 0,   s: 0, l: 0.5 }) // { r: 128, g: 128, b: 128 } (gray)
```

`rgbToHsl` и `hslToRgb` дают точное двустороннее преобразование (максимальная ошибка канала: 0 в пределах 8-битной точности).

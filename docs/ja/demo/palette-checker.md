# パレットチェッカー

カラーパレットがすべての色覚異常タイプの下で識別可能かどうかを確認します。色を入力するか、プリセットを選択して、任意のペア間の最小コントラスト比を確認してください。

<script setup>
import PaletteChecker from '../../.vitepress/theme/components/PaletteChecker.vue'
</script>

<ClientOnly>
  <PaletteChecker />
</ClientOnly>

## 結果の読み方

各行は特定の色覚異常タイプで変換されたパレットを表示します：

- **緑のバッジ（3:1以上）** — 色が容易に識別可能
- **赤のバッジ（2:1未満）** — 色が混同される可能性あり。パレットの調整を検討してください

比率はその行内の任意の2色間の**最小ペアワイズコントラスト**です。1:1の比率は2色が同一であることを意味します。

## デザインのヒント

### チャートとデータ可視化向け

- 隣接するデータ系列間で**少なくとも3:1のコントラスト**を使用
- 純粋な色相の変化よりも**輝度の変化**（明暗）を優先
- 二次的な差別化要素として**パターン、ラベル、アイコン**を追加
- まず**2型色覚異常（デューテラノマリー）**（最も一般的、男性の約5%）に対してパレットをテスト

### UIステータスカラー向け

- 成功/エラーに赤/緑だけに頼らない — アイコンやテキストを追加
- **緑 + 赤**の代わりに**青 + オレンジ**を使用してCVDコントラストを改善
- すべてのステータスカラーが色相だけでなく**明度**で異なることを確認

### 安全な色の組み合わせ

以下のペアはほとんどのCVDタイプで識別可能です：

| ペア | 効果的な理由 |
|---|---|
| 青 + オレンジ | 高い輝度コントラスト、異なる混同線 |
| 青 + 黄 | S錐体対L+M、3型色覚異常のみ問題 |
| 濃い青 + 薄いピンク | 輝度差がすべてのタイプで維持される |
| 黒 + 任意の明るい色 | 無彩色 + 有彩色は常に機能 |

## コード

```ts
import { simulate } from 'chromanopia'

const palette = ['#e63946', '#f4a261', '#2a9d8f', '#264653']
const type = 'deuteranopia'

const simulated = palette.map(color => simulate(color, type))
// → ['#886b1f', '#c4a833', '#7ba065', '#3b4e4f']
// 確認：これらの中に類似しすぎるものはないか？
```

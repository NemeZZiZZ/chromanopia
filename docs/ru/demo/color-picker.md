# Выбор цвета

Выберите любой цвет и посмотрите, как он выглядит при всех 8 типах нарушений цветового зрения. Меняйте модель и степень, чтобы изучить различия.

## Что вы видите

Каждый образец показывает результат `simulate(color, type, { model, severity })`:

- **Нормальное зрение** — исходный цвет без изменений
- **Протанопия / Протаномалия** — сниженная или отсутствующая реакция L-колбочек (красных)
- **Дейтеранопия / Дейтераномалия** — сниженная или отсутствующая реакция M-колбочек (зелёных)
- **Тританопия / Тританомалия** — сниженная или отсутствующая реакция S-колбочек (синих)
- **Ахроматопсия / Ахроматомалия** — полная или частичная потеря функции всех колбочек

## Попробуйте эти цвета

| Цвет | Почему это интересно |
|---|---|
| `#e63946` (красный) | Кардинально меняется для протан/дейтан типов |
| `#2a9d8f` (бирюзовый) | Трудно отличить от серого при дейтеранопии |
| `#f4a261` (оранжевый) | Путается с жёлтым при протан-нарушениях |
| `#6a4c93` (фиолетовый) | Путается с синим при дейтан/протан типах |
| `#00ff00` (чистый зелёный) | Почти невидим при дейтеранопии |

## Код

```ts
import { simulate } from 'chromanopia'

// The code behind each swatch:
const result = simulate('#e63946', 'protanopia', {
  model: 'machado',
  severity: 1,
})
// → '#886b1f'
```

# Модели симуляции

chromanopia реализует три рецензированные модели симуляции. Каждая по-своему балансирует между точностью и скоростью.

## Обзор

| Модель | Год | Метод | Скорость | Точность | Применение |
|---|---|---|---|---|---|
| **Viénot** | 1999 | Матрица 3×3, неотрицательная | Самая быстрая | Хорошая | UI в реальном времени, быстрый предпросмотр |
| **Machado** | 2009 | Матрица 3×3, спектральный сдвиг | Быстрая | Лучше | Выбор по умолчанию, лучший баланс |
| **Brettel** | 1997 | Попиксельная проекция CIE xyY | ~1.2× медленнее | Лучшая | Исследования, валидация, эталон |

## Viénot (1999)

**Статья:** Viénot, Brettel & Mollon — *Digital video colourmaps for checking the legibility of displays by dichromats*

- Все значения матрицы неотрицательны (≥ 0)
- Суммы строк равны 1.0 — яркость идеально сохраняется
- Самое быстрое вычисление: одно умножение матрицы на пиксель
- Менее точна для граничных случаев и частичных нарушений
- Лучше всего для: приложений реального времени, шейдерных конвейеров с ограниченным бюджетом

```ts
simulate('#e63946', 'protanopia', { model: 'vienot' })
```

## Machado (2009)

**Статья:** Machado, Oliveira & Fernandes — *A physiologically-based model for simulation of color vision deficiency*

- Использует спектральный сдвиг для моделирования изменений реакции колбочек
- Значения матрицы могут быть отрицательными и > 1 (экстраполяция)
- Физически точнее, чем Viénot
- Модель по умолчанию в chromanopia
- Лучше всего для: большинства задач, тестирования доступности, инструментов дизайна

```ts
simulate('#e63946', 'protanopia', { model: 'machado' })
// Это значение по умолчанию — эквивалентно:
simulate('#e63946', 'protanopia')
```

## Brettel (1997)

**Статья:** Brettel, Viénot & Mollon — *Computerized simulation of color appearance for dichromats*

- Проецирует каждый пиксель на линию смешения в цветовом пространстве CIE xyY
- Не одна матрица — каждый пиксель требует конвертации XYZ, проекции и обратного преобразования
- Самая точная модель, особенно для тританопии
- ~1.2× медленнее матричных моделей
- Для палочковой монохромазии (ахроматопсия / ахроматомалия) использует матрицы Machado
- Лучше всего для: исследований, эталонных результатов, валидации с другими реализациями

```ts
simulate('#e63946', 'protanopia', { model: 'brettel' })
```

::: tip
Используйте `isMatrixModel(model)`, чтобы проверить, возвращает ли модель матрицу 3×3. Это полезно при выборе между Sharp `recomb` (матричный путь) и обработкой сырого буфера (Brettel).
:::

## Варианты аномалий

Каждое нарушение колбочек имеет частичный вариант (аномалия):

| Полное нарушение | Частичный вариант | Затронутая колбочка |
|---|---|---|
| Протанопия | Протаномалия | L (красная) |
| Дейтеранопия | Дейтераномалия | M (зелёная) |
| Тританопия | Тританомалия | S (синяя) |
| Ахроматопсия | Ахроматомалия | Все колбочки (только палочки) |

Для матричных моделей симуляция аномалий использует интерполяцию между единичной матрицей и матрицей полного нарушения.

Для Brettel аномалии используют взвешенное смешение (фактор 1.75/2.75) между исходным и симулированным цветом дихромата.

## Палочковая монохромазия

Ахроматопсия (полная цветовая слепота) и ахроматомалия (частичная) — это не нарушения отдельных колбочек, а результат отсутствия или снижения функции колбочек в целом. Используются коэффициенты яркости Rec. 709:

```
R' = 0.2126·R + 0.7152·G + 0.0722·B
G' = 0.2126·R + 0.7152·G + 0.0722·B
B' = 0.2126·R + 0.7152·G + 0.0722·B
```

Эта матрица одинакова для всех трёх моделей, так как она не зависит от модели.

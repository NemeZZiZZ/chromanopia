# Проверка палитры

Проверьте, остаётся ли ваша цветовая палитра различимой при всех типах нарушений цветового зрения. Введите свои цвета, выберите пресет и посмотрите минимальный коэффициент контрастности между любой парой.

## Как читать результаты

Каждая строка показывает вашу палитру, преобразованную определённым типом нарушения:

- **Зелёный значок (≥ 3:1)** — цвета легко различимы
- **Красный значок (< 2:1)** — цвета могут быть спутаны; рассмотрите изменение палитры

Соотношение — это **минимальный попарный контраст** между любыми двумя цветами в строке. Соотношение 1:1 означает, что два цвета идентичны.

## Советы по дизайну

### Для графиков и визуализации данных

- Используйте **контраст не менее 3:1** между соседними рядами данных
- Предпочитайте **вариацию яркости** (светлый и тёмный) вместо чистой вариации оттенка
- Добавляйте **паттерны, подписи или иконки** как вторичный дифференциатор
- Сначала тестируйте палитру на **дейтераномалию** (наиболее распространённая, ~5% мужчин)

### Для цветов статусов в UI

- Не полагайтесь только на красный/зелёный для успеха/ошибки — добавляйте иконки или текст
- Используйте **синий + оранжевый** вместо **зелёный + красный** для лучшего контраста при CVD
- Убедитесь, что все цвета статусов различаются по **светлоте**, а не только по оттенку

### Безопасные цветовые комбинации

Эти пары остаются различимыми при большинстве типов CVD:

| Пара | Почему это работает |
|---|---|
| Синий + Оранжевый | Высокий контраст яркости, разные линии смешения |
| Синий + Жёлтый | S-колбочка против L+M, проблематично только для тританопии |
| Тёмно-синий + Светло-розовый | Разница в яркости сохраняется при всех типах |
| Чёрный + любой яркий цвет | Ахроматический + хроматический всегда работает |

## Код

```ts
import { simulate } from 'chromanopia'

const palette = ['#e63946', '#f4a261', '#2a9d8f', '#264653']
const type = 'deuteranopia'

const simulated = palette.map(color => simulate(color, type))
// → ['#886b1f', '#c4a833', '#7ba065', '#3b4e4f']
// Check: are any of these too similar?
```
